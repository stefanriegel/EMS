<objective>
Create the FeaturePipeline module that extracts training features from HA statistics and optionally InfluxDB, with a 1-hour cache for nightly batch efficiency.

Purpose: Centralizes feature extraction for all ML models (consumption forecaster, anomaly detector, tuner). Currently each model would need to independently query data sources. FeaturePipeline provides a single cached read so multiple models in the nightly batch share one I/O round.

Output: `backend/feature_pipeline.py` with FeaturePipeline class and `tests/test_feature_pipeline.py`.
</objective>
