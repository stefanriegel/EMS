1. **Define EntityDefinition dataclass** at module level (replace _ENTITIES tuple):

```python
@dataclass(frozen=True)
class EntityDefinition:
    entity_id: str        # e.g. "huawei_soc"
    name: str | None      # Short name or None (null name for binary_sensor with device_class)
    platform: str         # "sensor" or "binary_sensor"
    unit: str | None
    device_class: str | None
    state_class: str | None
    entity_category: str | None  # None, "diagnostic", or "config"
    value_key: str        # Field name in CoordinatorState or extra_fields
    device_group: str     # "huawei", "victron", or "system"
```

4. **Availability topic and LWT:**
   - Add `_AVAILABILITY_TOPIC = "ems/status"` constant
   - In `connect()`, before `self._client.connect(...)`, call:
     ```python
     self._client.will_set(self._AVAILABILITY_TOPIC, "offline", qos=1, retain=True)
     ```
   - In `_on_connect`, after setting connected, publish "online":
     ```python
     client.publish(self._AVAILABILITY_TOPIC, "online", qos=1, retain=True)
     ```
   - In `disconnect()`, before calling `self._client.disconnect()`, publish "offline":
     ```python
     self._client.publish(self._AVAILABILITY_TOPIC, "offline", qos=1, retain=True)
     ```
