## Interaction States

### Probe Flow (Steps 1-4)

1. **Idle**: "Test Connection" button enabled (if host field non-empty), no badge
2. **Testing**: Button text changes to "Testing...", button disabled, no badge
3. **Success**: Button re-enabled, green badge "Connection OK" appears inline
4. **Failure**: Button re-enabled, red badge "Failed: {reason}" appears inline
5. **Partial** (Victron only): Button re-enabled, amber badge with TCP-only warning

### Wizard Navigation

- **Step 1**: No Back button. Next button right-aligned.
- **Steps 2-5**: Back button left-aligned, Next button right-aligned.
- **Step 6**: Back button left-aligned. "Finish Setup" replaces Next.
- **Progress dots**: Current step = blue enlarged, completed = green, future = gray.

### Advanced Toggle (Step 2)

- **Collapsed** (default): Only host and port fields visible. Toggle reads "Advanced: Unit IDs".
- **Expanded**: Three additional fields appear (System ID, Battery ID, VEBus ID) with defaults pre-filled (100, 225, 227).
- Uses native `<details><summary>` -- no JavaScript state management.
